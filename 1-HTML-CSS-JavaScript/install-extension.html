How to install and Use VSCode's Live Server extension

Install | Use

Installation Instructions

  1. In Visual Studio Code click the Extensions icon in the left column toolbar
  2. In the search bar that appears type Live Server
  3. Select the Live Server extension. It should be the top choice.
  4. In the page for Live Server that appears click the green install button

Usage Instructins

  1. Visual Studio Codemust be open to a folder to use Live Server
  2. In VS Code's Explorer, right click (Windows) or Control+click (macOS) on an HTML file. From the context menu that appears, select Open with Live Server
